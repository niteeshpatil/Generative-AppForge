# Generative-AppForge
Skills: Flutter, Python, LLM, AI

- POC of an AI-powered tool using the LLM Gemini API to enhance real-time front-end mobile development with
voice and text prompts.
- Significantly accelerated frontend development, improving efficiency and productivity.

#  Intention
 - Here is one simple way of making use of LLM-powered AI, for software development in real-time,
   to make front-end mobile development faster and easier with voice or text prompting.
